# Predict-the-Next-Movie-Classic-Blockbuster-Text-Analytics-in-SAS-Model-Studio
**Predict the Next "Movie Classic" Blockbuster: Text Analytics in SAS Model Studio**

Step into the role of a new analyst at Movie Recommenders Unlimited, Inc. and learn how to use SAS Visual Text Analytics in SAS Model Studio to uncover patterns in film data. In this hands-on workshop, you’ll explore textual features from historical movies—such as descriptions, genres, and reviews—and build predictive models to determine which films have the potential to become the next “movie classic” blockbuster. Along the way, you’ll gain practical experience with natural language processing (NLP) techniques and see how text analytics can power real-world recommendation systems.
